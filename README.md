# Basic Calculator

A simple command-line calculator built with Python for beginners. This program takes two numbers and an arithmetic operator from the user, then performs the selected calculation.

## Features

* Addition (`+`)
* Subtraction (`-`)
* Multiplication (`*`)
* Division (`/`)
* Modulus (`%`)
* Displays an error message for invalid operators

## Technologies Used

* Python 3

## How to Run

1. Make sure Python is installed on your computer.
2. Run the program:

   ```bash
   python basiccalculator.py
   ```
3. Enter:

   * First number
   * Second number
   * Operator (`+`, `-`, `*`, `/`, `%`)
4. The program will display the result.

## Purpose

This project was created to practice Python fundamentals, including:

* User input
* Variables
* Conditional statements (`if`, `elif`, `else`)
* Basic arithmetic operations
